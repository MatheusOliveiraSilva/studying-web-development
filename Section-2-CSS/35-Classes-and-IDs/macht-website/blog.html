Post #1 how to use special characters in HTML

Sometimes we need to use some special characters that are part of the HTML syntax, to solve this problem we can use the HTML entities. The table below shows some kinds of special characters.

Char Entity Description
& &amp; Ampersand
< &lt; Less Than
> &gt; Greater Than
≤ &le; Less Equals
≥ &ge; Greater Equals
£ &pound; Pound
¥ &yen; Yen
¢ &cent; Cent
€ &euro; Euro
® &reg; Registered trade mark

Well, in table above we saw some of the HTML entities, has much more than this that we saw here, to see the complete table access this link: https://www.w3schools.com/charsets/ref_html_entities_4.asp .
